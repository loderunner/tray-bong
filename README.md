# Tray-Bong

> The "tray bong gang" (from the French _"très bon"_, meaning "very good") was a
> name given by the Digswell locals to the workers of the Digswell Viaduct who,
> having returned from a job in France, had picked up French words, albeit
> imperfectly.
>
> see
> [Eileen Wallace — _Children of the Working Poor_](https://books.google.fr/books?id=0RaCcx3U_IAC&printsec=frontcover&hl=fr#v=onepage&q=tray%20bong&f=false)
